                        API ENDPOINTS (7 TOTAL)
================================================================================

GET    /health                          - Health check
POST   /auth/signup                     - Create account
POST   /auth/signin                     - Login
GET    /auth/me                         - Get current user
GET    /auth/user/:userId               - Get user by ID
POST   /auth/signout                    - Logout
PUT    /auth/user/:userId/background    - Update background

================================================================================
                         PERSONALIZATION FLOW
================================================================================

User Action:
  1. User signs up with software/hardware background
  2. User asks question in chatbot
  3. Frontend sends user_id with query
  4. Backend fetches user background from database
  5. System prompt enhanced with personalization
  6. OpenAI receives tailored instructions
  7. Response is optimized for user's experience level

Integration:
  - Python receives user_id
  - Queries Neon database for background
  - Injects into system prompt
  - OpenAI uses personalized context
  - More relevant and appropriate responses
